【任务书】

1. 需求说明：
   将动作库前端三个角色的页面从 mock 数据切换为真实后端接口，
   并补全缺失功能：视频播放、分页、步骤动态增删、驳回原因展示、
   管理员 Tab 切换与详情预览。

2. 实现思路：
   - api 层：删除 mock，改为 axios 真实请求
   - 用户端：列表加分页，详情加 <video> 播放器
   - 教练端：步骤/注意事项改为动态列表，我的投稿加驳回原因展示和删除
   - 管理员端：加 Tab（待审核/已上线/已驳回/已下架），加详情预览弹窗，加下架按钮

3. 前端要做什么：
   - src/api/action.js     → 全部改为真实 axios 请求
   - src/api/admin.js      → 动作相关接口改为真实请求
   - user/ActionLibrary/index.vue  → 加分页、封面图展示
   - user/ActionLibrary/Detail.vue → 加 <video> 播放、cautions 改为列表渲染
   - coach/ContentPublish/ActionPublish.vue → 步骤/注意事项动态增删、驳回原因、删除
   - admin/ContentReview/ActionReview.vue  → Tab 切换、详情预览、下架

4. 后端要做什么：已完成，接口地址如下。

5. 接口对应关系：

   | 前端调用函数         | 后端接口                              | 说明               |
   |---------------------|---------------------------------------|--------------------|
   | getActions(params)  | GET  /api/actions                     | 用户端列表         |
   | getActionDetail(id) | GET  /api/actions/{id}                | 用户端详情         |
   | publishAction(data) | POST /api/coach/actions               | 教练发布           |
   | getMyActions()      | GET  /api/coach/actions               | 教练我的投稿       |
   | updateMyAction(id)  | PUT  /api/coach/actions/{id}          | 教练修改被驳回     |
   | deleteMyAction(id)  | DELETE /api/coach/actions/{id}        | 教练删除           |
   | getAdminActions()   | GET  /api/admin/actions?status=       | 管理员列表         |
   | reviewAction(id)    | PUT  /api/admin/actions/{id}/review   | 管理员审核         |
   | offlineAction(id)   | PUT  /api/admin/actions/{id}/offline  | 管理员下架         |

6. 数据表结构：无需变更。
